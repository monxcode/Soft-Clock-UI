# ⏰ Pastel Clock & Calendar Widget

A beautifully designed **pastel-themed clock and calendar widget** featuring a dynamic background, soft transitions, and a fully responsive user interface.  
This widget focuses on minimalism, smooth animations, and a pleasant visual experience.

---

## ✨ Features
- Real-time digital clock
- Live date and calendar display
- Soft pastel color theme
- Dynamic background effects
- Smooth transitions and animations
- Fully responsive UI (mobile & desktop)

---

## 🎨 Design Philosophy
- Minimal and aesthetic UI
- Soft pastel colors for eye comfort
- Clean typography
- Subtle motion for a modern feel

---

## 🛠️ Technologies Used
- **HTML** – Structure  
- **CSS** – Pastel styling, transitions, responsiveness  
- **JavaScript** – Real-time clock and date logic  

---

## 📱 Development Setup
Built entirely using **mobile-first development tools**, proving efficient UI creation without desktop dependency:
- **Acode** – Mobile code editor  
- **Responsive-first approach**

---

## 📸 Preview

## 🚀 Usage
1. Open the widget in any modern browser  
2. The clock updates automatically in real time  
3. Date and calendar adjust dynamically  
4. Enjoy a calm and aesthetic interface
